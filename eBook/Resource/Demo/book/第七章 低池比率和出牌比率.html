第七章 低池比率和出牌比率

下面了解熟悉在turn 和rivier 时的下注策略, 并继续在实战中练习。

掌握彩池比率和出牌比率非常重要。如果你的起手牌很好，翻牌出得也还不错，那么转牌（turn）和河牌（river）显然你是要继续玩下去。如果翻牌出来和你的牌相差很远，你当然就会盖牌。但是如果你的牌有成为一付好牌的可能，但是现在还不怎么样，那么你是否应该继续玩下去呢?扑克是一个机会游戏，所以一个好的扑克玩家在是否继续玩下去的时候应该考虑概率问题，彩池比率的概念就是用来比较你成手的几率和赔付的比率，以帮助你决定是否继续下去。

计算池底比率很简单，首先，看一下现在在彩池的筹码数量，包括现在以有的彩池和你认为继续下注可能跟的注(这种可能的赔率叫预期赔率implied odds)。然后把这个数字除以你的要下注的筹码。例如，彩池有$6，你需要跟$1，所以彩池比率就是1:6，如果有玩家对你加注了，你就要重新计算彩池比率，这样，你的除数就是你先下的注加你加注的金额。

计算彩池比率的目的是帮助你决定是否等待一手你可以赢的牌。下表显示的是你的牌在转牌和河牌变好或者成手的几率，是根据还没有出现的牌的数目进行计算的。例如，如果在翻牌已经有了四张牌的同花黑桃，那么你还需要等一张黑桃，没有出现的黑桃有9张。如果你手上有A，K，你认为只需要来一张A或K你就可以必胜，那么你要等的牌就还有6张。

然后就需要把出牌比率和彩池比率进行比较。如果出牌比率高过彩池比率(这里的高低是指被除数，比如1:4高过1:2)，那么你潜在的赔付就比较不好。举例来说，彩池有$8，你需要$1来叫牌，彩池比率是1:8。 你手上的牌是梅花5，黑桃6，而翻牌是梅花8和方块9，以及黑桃J，你有的是一个中张顺子，你的成手需要的牌只有四张，所以出牌几率基本上是1:12(52减去5张等于47，4比47基本上是1:12)， 这就大于彩池比率的1:8了，那么就应该盖牌。另外一个例子是:同样彩池比率1:8，你已经有了四张同花牌，你等的牌还有9张，那么你的出牌比率是1:5，那么就很适合跟注或者加注了。

下面再举一个简单的例子，说明这些概率知识的重要性：

转牌翻开以后，你有四张红桃，你完成红桃同花牌型概率为9比47，大约为5分之1，你相信同花会赢这轮牌，而这时的总赌注为$200，你只要再跟注$20就可以看到河牌，总注概率为20比200，即为十分之一，这就是说，如果出现5次这种情况，你赢1次输4次，你赢一次的钱为$200，而你输4次的钱总数为$80，你1次赢钱的总数远大于你4次输钱的总数，因此这种情况下你一定要跟牌而不是弃牌。 反过来，如果这时的总赌注为$60，而你要跟注$20才可以看到河牌，总注概率为20比60，即为3分之1，这就是说，如果出现5次这种情况，你赢1次输4次，你赢一次的钱为$60，而你输4次的钱总数为$80，你一次赢钱的总数小于你4次输钱的总数，因此这种情况下你一定要弃牌而不是跟牌。

计算FLOP之后拿牌的概率

flop之后，有47张玩家没有看到的牌，turn之后，还有46张。所以你有两次机会可以拿到你期望要的牌。用下面这个公式来计算拿牌的概率：
% = 100 [ 1 - (47 - N) (46 - N) / (47 x 46)] = 100 [ 1 - (47 - N) (46 - N) / 2162]
其中：
=% = 在后两张牌里拿到期望牌的概率(turn & river)
N = flop之后期望牌的个数

计算turn之后的概率：

turn之后，只有46张牌玩家没有看到，所以概率的计算公式为：
% = 100 ( M / 46)
其中：
% = 在最后一张牌拿到期望牌的概率(river)
M = turn之后期望牌的个数

概率的与Odds的换算：

Odds = % : (100 - %)
例如：如果你在flop拿到flush draw（差一张flush),那么你的期望牌为9张，在river拿到一张同花的概率为19.6%
Odds = 19.6 : (100 - 19.6) = 1 : 4.1

一些有用的概率/Odds：

--底牌为一对，flop拿到3张的可能为：12% 或者 1:8
--底牌为AK,flop拿到至少一张A或者K的可能为: 32% 或者 1:2
--底牌为两张同花，flop另外两张同花的可能为：11% 或者 1:8
--桌面上三张同花，你手里为Kx同花，此时输给Ax同花的可能为6%,或者16:1对你有利
--拿到两张同花的底牌的概率为23%或者1:3
